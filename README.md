# Hockey Stats
An old WPF application I quickly created to retrieve hockey players statistics/per team via API calls.

## Goal
Get team player stats.

## Technologies
- WPF/MVVM
- JSON

## How-To
1. Run project.
2. Click on the **(Re)Load Teams** button to populate combo boxes.
3. *Navigate*.
